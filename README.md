# 🐱🔍 Mystères à Chaville

Jeu éducatif d'enquêtes pour les 9-12 ans : observe, déchiffre, calcule, déduis !

- `index.html` — menu d'accueil
- `academie.html` — le jeu principal (carte de Chaville, 6 enquêtes)
- `demo.html` — démo aventure (déplacement du détective, pixel art)
- PWA installable : sur téléphone, Chrome → ⋮ → « Ajouter à l'écran d'accueil »
